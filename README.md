## Install

```sh
bundle && yarn
```

Run migrations

```sh
rake db:migrate
```

Run seeds

```sh
rake db:seed
```

## Usage

Create a `.env` file with the following variables:

```
DB_PASS=...
DB_USER=...
SENDGRID_API_KEY=...
```

Run the server

```sh
rails s
```

## Run tests
